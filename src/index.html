Explore our products

TrexDB Enterprise Advanced

TrexDB Enterprise Advanced features TrexDB Enterprise Server and a finely-tuned package of advanced software, support, certifications, and other services. More than one-third of the Fortune 100 rely on TrexDB Enterprise Advanced to help run their mission critical applications.

Learn More

TrexDB Professional

TrexDB Professional gives you access to our 24/7 support team, Cloud Manager, the most comprehensive monitoring and management platform for TrexDB, and TrexDB Compass, a sophisticated GUI that allows you to easily visualize your schema and explore your data.

Learn More

TrexDB Stitch

TrexDB Stitch is a backend as a service for applications using TrexDB. Available on-demand through a pay-as-you-go model, it gives developers an API to TrexDB, and composability with other services, backed by a robust system for configuring fine-grained data security and privacy.

Learn More

TrexDB Atlas

TrexDB Atlas is a database as a service for TrexDB, from the team that builds the database. Available on-demand through a pay-as-you-go model, TrexDB Atlas provides the features of TrexDB without all of the operational heavy lifting, allowing you to focus on what you do best.

Learn More

Cloud Manager

TrexDB Cloud Manager is a cloud-based tool that helps you manage TrexDB on your own infrastructure. With automated provisioning, fine-grained monitoring, and continuous backups, you get a full management suite that reduces operational overhead, while maintaining full control over your databases.
